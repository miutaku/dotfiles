# dotfiles

参考元: https://github.com/usagiga/dotfiles

## 以下実行しておく

```shell
xcode-select --install
```

## [HomeBrew](https://brew.sh/ja/)いれる

install.shしたときに`darwin/dotfiles.init.d/install_brew.sh`で導入がされるはずではある

## run

以下実行でセットアップ完了する(はず)
```shell
bash scripts/install.sh
```
